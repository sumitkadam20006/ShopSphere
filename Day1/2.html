a) Difference between a Client and a Server

->Client: A device or application that requests data or services . Server: A computer or system that provides data or services to clients.

b) What does DNS do, and why would the internet not work without it?

->DNS (Domain Name System) translates website names like www.google.com into IP addresses. Without DNS, users would have to remember and enter IP addresses to visit websites.
